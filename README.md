# 第8章完整交互式数据可视化平台

## 📊 项目概述

这是一个基于第8章 Jupyter Notebook 创建的完整交互式数据可视化平台，包含了教材中所有的图形类型。该平台具有丰富的交互功能，支持实时数据编辑、主题切换、图表导出等功能。

## 🎯 核心特色

### 📈 **11种专业图表类型**
| 图表类型 | 功能描述 | 应用场景 |
|---------|---------|---------|
| 📈 **折线图** | 时间序列分析 | 趋势分析、时序数据 |
| 🔵 **散点图** | 相关性分析 | 变量关系、数据分布 |
| 📊 **直方图** | 数据分布分析 | 频次统计、分布特征 |
| 📦 **箱线图** | 统计分布分析 | 中位数、四分位数、异常值 |
| 🔥 **热力图** | 相关性矩阵 | 相关性分析、密度分布 |
| 🗺️ **等高线图** | 三维地形数据 | 地理数据、函数可视化 |
| 🌐 **3D曲面图** | 立体数据展示 | 多维数据、函数曲面 |
| 🌿 **棉棒图** | 数据对比分析 | 轿车燃料消耗对比 |
| 📅 **甘特图** | 项目时间线管理 | 项目进度、任务调度 |
| 👥 **人口金字塔** | 人口结构分析 | 年龄性别分布、人口统计 |
| ⚖️ **哑铃图** | 数据对比展示 | 变化对比、前后差异 |

## 🎯 使用方法

### 💻 **本地运行**
1. 双击 `index.html` 或 `chapter8_complete_interactive_visualization.html`
2. 在浏览器中打开即可使用所有功能

### 🌐 **GitHub Pages 部署**
1. 使用 `deploy.bat` 脚本一键部署
2. 或手动上传到 GitHub 仓库
3. 在 Settings → Pages 中启用 GitHub Pages
4. 选择 main 分支和 /(root) 文件夹

### 📱 **移动端使用**
- 完全响应式设计，支持手机、平板、电脑
- 触摸友好的交互界面
- 自适应图表布局
